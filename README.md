# Restaurant Management System

## توضیحات پروژه
سیستم مدیریت رستوران به زبان **Java** برای مدیریت **کارمندان، مشتریان، منو و سفارش‌ها**.  
این پروژه از مفاهیم **OOP** مثل **ارث‌بری، کپسوله‌سازی و متدهای انتزاعی** استفاده می‌کند.

## امکانات
- **مدیریت اعضا:** کارمندان و مشتریان با ویژگی‌های خاص و عمومی، محاسبه حقوق و امتیاز وفاداری  
- **مدیریت منو:** غذاها و نوشیدنی‌ها با جزئیات مانند تندی، زمان آماده‌سازی، اندازه و دما  
- **سفارش‌ها:** ثبت سفارش، محاسبه جمع کل و اعمال تخفیف، نمایش خلاصه سفارش  

## نحوه اجرا
- پروژه را در IDE باز کرده و کلاس `Main` را اجرا کنید.  
- خروجی شامل ایجاد کارمندان و مشتریان، اضافه کردن آیتم‌های منو، ثبت سفارش و نمایش اطلاعات است.

## نکات
- استفاده از **Abstract Classes** و **Method Overriding**  
- مدیریت شناسه‌های خودکار: `customerId`, `employeeId`, `orderId`  
- استفاده از **ArrayList** برای ذخیره منو و سفارش‌ها
